# WT-Project

#Hosting The Server
Clone the repo using ```git clone https://github.com/akashchekodu/WT-Project.git``` or download the zip file and unzip
then
```
cd backend
npm i
npm start
```
This will start the server in port  7000

#Loading the front-end

(Download the files)
then
```
cd front-end
npm i
npm start
```
